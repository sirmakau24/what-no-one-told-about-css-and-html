Semantic HTML

Semantic HTML uses meaningful tags to describe the structure and purpose of content, improving accessibility and SEO.

Common Semantic Tags

  • <header>: Introductory content or navigation links
  • <nav>: Navigation links
  • <main>: Main content of the document
  • <section>: Thematic grouping of content
  • <article>: Self-contained composition (e.g., blog post)
  • <aside>: Content related to the main content (e.g., sidebar)
  • <footer>: Footer for a section or page
  • <figure> and <figcaption>: Media and its caption
  • <mark>: Highlighted text
  • <time>: Date/time value

  • Accessibility: Semantic tags help screen readers and assistive technologies.

      /* Flexbox container */

    background: #f0f0f0;

  }

  .grid-container {

    gap: 10px;
  }
  .grid-container img {
    width: 100%;
    border-radius: 6px;
  }

  /* Responsive example */
  @media (max-width: 600px) {
    .flexbox-container {
    flex-direction: column;
    }
    .grid-container {
    grid-template-columns: 1fr;
    }

    Related Concepts

    • Positioning types: static, relative, absolute, fixed, sticky
    • Transitions: Smoothly animate property changes (e.g., background-color, transform)
    • Animations: Use @keyframes for complex effects
    • Z-index: Controls stacking order of positioned elements
    • Use cases: Tooltips, modals, sticky headers, animated buttons
      /* Positioning */
  .relative-box {
    position: relative;
  }
  .absolute-box {
    position: absolute;
    top: 10px;
    left: 10px;
  }
  .sticky-box {
    position: sticky;
    top: 0;
  }

  /* Animation */
  @keyframes bounce {
    0%, 100% { transform: translateY(0); }
    50% { transform: translateY(-20px); }
  }
  .animated-box {
    animation: bounce 2s infinite;
  }

  /* Transition */
  .btn-transition {
    transition: all 0.3s ease;
  }
  .btn-transition:hover {
    background-color: #ff4081;
    color: white;
    transform: scale(1.1);
